About this project

This project is created by Aman Sinha. The main motive of this project is to simplify the syntex of HTML and convert the language in user friendly English statements.
Using the set of statements now anyone can create a website without using all opening and closing tags of html.

Features of using this language

  • Create Website using simple statements
  • Statements also support css
  • More fast and easy approach
  • No code Development
  • Support Many HTML-5 tags

Rules or statements structure to use this language

  1. Structure:

    Start the statement with "$" sign.
    Then write the tag name followed by the content
    of the tag in round brackets "()"
